Install

Prerequisites

Step 1: venv

Create a venv using python 3.7, a higher version is not supported at the time by the keras and tenserflow package.

step 2: install

Use 'pip install -r requirements'

Warning:
If you are on windows then use a wheel to intall numpy and sci-py before using pip install.

After install

After install run 'flask run'
Then using a browser surf to localhost:5000/api/init and rerun 'flask run'
This will download every needed corpus file.
